// 기능

1. 회원관리 (가입, 탈퇴, 로그인, 로그아웃) : loginForm(초기페이지)와 햄버거 메뉴 이용


2. 레시피
- 레시피 등록 : 냉장고의 레시피쓰기나 햄버거메뉴의 레시피쓰기버튼
--> ** 레시피 재료 추가 시 재료 검색버튼을 클릭하여 재료 선택후 amount와 unit을 입력하고 add버튼 클릭해야함 **
--> 조리과정은 '추가' 버튼 클릭하여 행 추가 후 차례로 과정 작성
( 자신이 직접 업로드한 이미지는 외부 브라우저에서는 보안상의 이유로 확인불가, 이클립스 브라우저에서만 가능 )

- 레시피 삭제 (자신이 작성한 것만 삭제 가능) : 냉장고의 마이레시피나 레시피 상세보기 페이지의 delete버튼

- 레시피 후기 작성 : 레시피 하단의 댓글창 이용

- 레시피 후기 확인 : 레시피 하단에서 확인가능

- 레시피 후기 삭제 (자신이 작성한 것만 삭제 가능) : 레시피 댓글창의 delete버튼

- 레시피 즐겨찾기 : 레시피 제목 하단의 달 모양 이모티콘 클릭 --> 냉장고의 Favorites 항목에서 즐겨찾기 목록 확인가능
--> Favorites 항목으 delete버튼을 클릭하여 즐겨찾기 해제 가능


3. 검색
- 레시피명으로 검색 : 상단의 검색창 이용 (김치, 계란 등 검색), 검색결과에서 more 클릭하여 상세페이지 확인가능

- 내가 보유한 재료가 포함된 레시피 검색 : 냉장고에서 Searching by My Ingredients 버튼 클릭


4. 음식 몇인분 계산
- 레시피 상세페이지에서 인원수 변경 후 적용버튼 클릭


5. 냉장고
- 재료 추가
1) MyIngredients 하단의 " + " 버튼 클릭
2) 검색 버튼 클릭하여 재료 검색 수행
3) 원하는 재료를 선택한 후 amount(양), unit(단위), expire date (유통기한) 입력하고 Add버튼 클릭
4) 1~3의 과정을 반복하여 여러개의 재료를 한번에 추가 가능
5) 원하는 재료를 모두 추가했으면 '재료 선택 완료' 버튼 클릭

- 재료 삭제 : 냉장고에서 재료 옆 delete버튼 클릭

- 유통기한 임박 재료 확인 : 냉장고의 Expired Ingredients 에서 확인 및 재료삭제 가능
